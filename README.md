📦 설치 및 실행


ARAM_정산기.html 파일을 받아 아무 폴더에 저장
파일을 더블클릭 → 브라우저에서 열림 (Chrome 또는 Edge 권장)
첫 실행 시 인터넷 연결 필요 (라이브러리 자동 다운로드, 이후 오프라인 가능)



🚀 시작하기

1단계 - 방 만들기


새 방 만들기 클릭 → 자동으로 방 코드 생성 후 입장
또는 원하는 방 코드를 직접 입력 (영문/숫자 4자 이상)
방 코드는 같은 기기에서 세션을 구분하는 용도 (친구 PC와 자동 동기화 X)


2단계 - 참가자 등록

참가자 탭 이동 → + 참가자 추가 → 닉네임 입력 → 추가

3단계 - 배팅금액 설정

참가자 탭 → 배팅금액 설정


1인당 배팅 기준 (예: 2,000원 설정 시 → 이기면 +2,000원, 지면 -2,000원)
프리셋(1천/2천/3천/5천) 클릭 또는 직접 입력 가능
✅ 균등 팀(예: 3vs3)이면 항상 딱 떨어지는 숫자로 계산됨



🎮 게임 기록하기


게임 탭 → + 게임 추가 클릭
배팅금액 선택 (기본값은 위에서 설정한 금액)
각 참가자의 A/B 버튼 클릭해서 팀 배정

불참한 멤버는 배정하지 않으면 됨 (중간에 나간 경우)



팀 A 승 🏆 또는 팀 B 승 🏆 선택
저장 클릭



판 삭제: 게임 기록 우측 ✕ 버튼




💰 정산 확인

정산 탭에서 확인

항목내용개인 수지초록색 = 받을 돈 / 빨간색 = 줄 돈송금 목록누가 → 누구한테 얼마 보낼지 (최소 거래수로 계산)


🔗 친구 PC에서 보는 방법 (공유)

데이터는 이 기기 브라우저에만 저장되므로, 친구에게 공유하려면 아래 방법 사용

내보내기 (내 PC → 친구 PC)


헤더 📤 공유 버튼 클릭
내보내기 탭 선택
📋 클립보드 복사 클릭
카카오톡 / 디스코드 등으로 코드 전송


가져오기 (친구 PC에서)


친구한테서 받은 코드 복사
앱에서 📤 공유 → 가져오기 탭
코드 붙여넣기 (Ctrl+V)
📥 불러오기 클릭



🔄 세션 관리


초기화: 정산 탭 하단 → 게임 기록 초기화 (참가자는 유지)
다음 게임 세션: 새 방 코드 생성 or 방 코드 입력 화면에서 기존 세션 클릭
데이터 보관: 브라우저 캐시 삭제 전까지 자동 유지



❓ 자주 묻는 질문

Q. 같은 방 코드 써도 친구 PC랑 공유 안 되나요?

A. 안 됩니다. 데이터는 각자 기기에 저장됩니다. 위의 공유 코드 기능을 사용하세요.

Q. 숫자가 소수점으로 나와요

A. 배팅금액이 팀 인원수로 안 나눠질 때 발생합니다. 균등팀(n:n)에서는 항상 딱 떨어집니다.

(예: 3인팀에서 1인당 2,000원 → ±2,000원으로 깔끔하게 계산됨)

Q. 인터넷 없이 쓸 수 있나요?

A. 한 번 열었던 PC에서는 브라우저 캐시로 오프라인 사용 가능합니다. (처음은 인터넷 필요)

Q. 데이터가 갑자기 사라졌어요

A. 브라우저 캐시 삭제나 시크릿 모드 사용 시 데이터가 없어질 수 있습니다.

중요한 세션은 공유 코드로 백업해두세요.
